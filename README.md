# ♟️ Chess System

### A console-based chess game built with Java using Object-Oriented Programming principles

---

## 🧭 Overview

**Chess System** is a Java application that simulates a complete chess match in the console.

The project focuses on applying **Object-Oriented Programming (OOP)** concepts and implementing real game rules such as piece movement, check, checkmate, and special moves.

---

## ⚙️ Tech Stack

- ☕ Java
- 🧠 Object-Oriented Programming (OOP)

---

## 🏗️ Architecture & Concepts

The project is structured to reinforce strong OOP practices:

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction (abstract classes and methods)
- Exception handling
- Layered organization (board, pieces, game logic, UI)

---

## ♜ Features

- Full chess game logic:
  - Piece movement validation
  - Turn-based system
  - Check and checkmate detection
- Special moves:
  - Castling
  - En passant
  - Pawn promotion
- Tracking captured pieces
- Console-based board rendering
